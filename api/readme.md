# API

## 1. 설치방법

### 1. Python 3.8버전 이상 설치한다.   

### 2. cmd 진입 후 가상환경을 생성한다.   
```
    python -m venv {가상환경 이름}
```
### 3. 가상환경을 실행합니다.
```
    {가상환경 이름}\scripts\activate
```
### 4. requirements.txt 패키지를 모두 설치한다.
```
    pip install -r requirements.txt
```
