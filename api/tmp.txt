department - 과와 관련된 모든 것
과사무소 위치
- department/office
기본호출 -> 모든 과 사무실 확인
- department/office/{과 이름}
원하는 과 사무실 위치 확인
- department/notice/{과 이름}
해당 과의 공지사항을 스크래핑해서 보여줌
- department/notice/kangwon
학교 홈페이지 학사공지 스크래핑해서 보여줌
- department/professor/{과 이름}
해당 과의 교수 목록 보여줌
- department/professor/{과 이름}/{교수 이름}
해당 교수의 정보 보여줌 (전화번호, 방 번호, 이메일 등)
