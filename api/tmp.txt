비슷한 이름은 어떻게 검색하나...?

department - 과와 관련된 모든 것
과사무소 위치
- department/office
기본호출 -> 모든 과 사무실 확인
- department/office/{과 이름}
원하는 과 사무실 위치 확인
- department/notice/{과 이름}
해당 과의 공지사항을 스크래핑해서 보여줌
- department/professor/{과 이름}
해당 과의 교수 목록 보여줌



cafeteria - 식당과 관련된 모든 것
식당
- cafeteria
기본호출 -> 모든 식당 확인
- cafeteria/{기숙사식당이름}
해당 식당의 메뉴 표시(아침, 점심, 저녁)
- cafeteria/{학식식당이름}
해당 식당의 메뉴 표시(판매 메뉴)
- cafeteria/{카페이름}
해당 카페의 메뉴 표시
